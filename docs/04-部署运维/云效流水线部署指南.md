# 云效流水线部署指南

## 快速部署

本项目使用阿里云云效流水线进行自动化部署，只需推送代码即可自动部署到腾讯云Lighthouse服务器。

## 环境变量配置

在云效流水线中配置以下环境变量：

### 服务器配置
- `SERVER_HOST`: 腾讯云Lighthouse服务器IP
- `SERVER_PASSWORD`: 服务器登录密码

### 数据库配置
- `MONGO_HOST`: MongoDB服务器地址（如：`10.0.20.14`）
- `MONGO_PORT`: MongoDB端口（如：`27017`）
- `MONGO_PASSWORD`: MongoDB密码

## 部署流程

1. **代码推送** → 推送到GitHub main/master分支
2. **自动构建** → 云效拉取代码并构建
3. **自动部署** → 部署到腾讯云服务器
4. **健康检查** → 验证服务状态
5. **部署完成** → 发送成功通知

## 访问地址

- 前端：`http://SERVER_HOST`
- 后端API：`http://SERVER_HOST:8002`
- 健康检查：`http://SERVER_HOST:8002/health`

## 回滚

如果部署失败，可在云效控制台使用内置回滚功能。
