# Easy Book 安装和启动指南

## 系统要求

- Windows 10/11
- Python 3.11+ （如果没有安装，请从 https://python.org 下载）
- Node.js 18+ （如果没有安装，请从 https://nodejs.org 下载）
- Docker Desktop （可选，用于运行MongoDB）

## 安装步骤

### 1. 检查Python环境

打开命令提示符（CMD）或PowerShell，运行：

```bash
python --version
```

如果显示版本号（如 Python 3.11.x），说明Python已安装。

### 2. 检查Node.js环境

```bash
node --version
npm --version
```

应该显示版本号（如 v18.x.x）。

### 3. 启动MongoDB数据库

#### 方式一：使用Docker（推荐）

1. 安装Docker Desktop
2. 运行以下命令启动MongoDB：

```bash
docker run -d --name mongodb -p 27017:27017 mongo:6
```

#### 方式二：使用本地MongoDB

1. 从 https://www.mongodb.com 下载并安装MongoDB Community Server
2. 启动MongoDB服务

### 4. 手动启动服务（推荐初次使用）

#### 启动后端

```bash
# 进入后端目录
cd D:\workspace\python\easy-book\backend

# 安装Python依赖
pip install -r requirements.txt

# 启动后端服务
python -m app.main
```

如果看到类似以下输出，说明后端启动成功：
```
INFO:     Uvicorn running on http://0.0.0.0:8000
INFO:     Application startup complete.
```

#### 启动前端（新开一个命令窗口）

```bash
# 进入前端目录
cd D:\workspace\python\easy-book\frontend

# 安装Node.js依赖
npm install

# 启动前端服务
npm run dev
```

如果看到类似以下输出，说明前端启动成功：
```
  VITE v5.0.0  ready in 323 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

### 5. 访问系统

打开浏览器，访问：http://localhost:5173

## 常见问题

### 问题1：Python命令不存在

**解决方案：**
1. 确保Python已正确安装
2. 检查环境变量PATH是否包含Python路径
3. 尝试使用 `py` 命令代替 `python`

### 问题2：pip安装依赖失败

**解决方案：**
```bash
# 升级pip
python -m pip install --upgrade pip

# 使用国内镜像源
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/
```

### 问题3：npm安装依赖失败

**解决方案：**
```bash
# 清除npm缓存
npm cache clean --force

# 使用国内镜像源
npm config set registry https://registry.npmmirror.com/
npm install
```

### 问题4：MongoDB连接失败

**解决方案：**
1. 确保MongoDB服务正在运行
2. 检查端口27017是否被占用
3. 如果使用Docker，确保容器正在运行：
   ```bash
   docker ps
   ```

### 问题5：端口被占用

**解决方案：**
1. 查看端口占用：
   ```bash
   # 查看8000端口（后端）
   netstat -ano | findstr :8000
   
   # 查看5173端口（前端）
   netstat -ano | findstr :5173
   ```

2. 结束占用进程：
   ```bash
   # 其中PID是上一步查到的进程ID
   taskkill /PID <PID> /F
   ```

## 使用一键启动脚本

如果所有环境都已正确安装，可以使用一键启动脚本：

```bash
# Windows
start.bat

# Linux/Mac
./start.sh
```

**注意：** 一键启动脚本会自动安装依赖，但如果环境变量配置有问题，仍可能失败。建议初次使用时手动启动。

## 验证安装

1. 后端验证：访问 http://localhost:8000/docs ，应该看到API文档页面
2. 前端验证：访问 http://localhost:5173 ，应该看到系统界面
3. 功能测试：添加一个学员，创建预约，测试签到功能

## 开发模式

如果需要进行开发，建议使用以下命令：

```bash
# 后端开发模式（自动重载）
cd backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 前端开发模式
cd frontend
npm run dev
```

## 停止服务

- 手动启动：直接关闭对应的命令窗口
- 一键启动：按任意键自动停止
