## ⚠️ 重要提醒

- **本地优先原则**: 所有修改必须先在本地验证再部署
- **备份先行**: 部署前务必备份现有数据
- **逐步验证**: 每个步骤完成后都要进行验证
- **文档同步**: 遇到问题请及时更新文档
