# Easy Book - 泳课学员管理系统技术选型文档

## 项目概述

Easy Book是一个轻量级的泳课学员管理系统，以学员为中心，提供预约管理、签到扣费、学员信息管理等功能。系统设计简洁，定位为个人工具级别的应用。

## 技术选型

### 前端技术栈

| 技术 | 版本 | 选择理由 |
|------|------|----------|
| **Vue.js** | 3.x | 当前最新稳定版本，性能优秀，生态成熟 |
| **Vite** | 4.x | 快速的构建工具，开发体验好，HMR速度快 |
| **Element Plus** | 2.x | 饿了么团队出品，Vue3生态最成熟的UI库 |
| **Pinia** | 2.x | Vue3官方推荐的状态管理库，API简洁 |
| **Vue Router** | 4.x | Vue官方路由管理器，支持Vue3 |
| **Axios** | 1.x | 成熟的HTTP请求库，支持拦截器和Promise |

### 后端技术栈

| 技术 | 版本 | 选择理由 |
|------|------|----------|
| **FastAPI** | 0.104+ | Python现代Web框架，自动生成API文档，开发效率高 |
| **MongoDB** | 6.x+ | NoSQL文档数据库，数据结构灵活，适合快速迭代 |
| **PyMongo** | 4.x+ | MongoDB官方Python驱动，简单直接，性能优秀 |
