# Easy Book 快速开始指南

> 🚀 快速搭建和运行Easy Book泳课学员管理系统

## 📋 系统要求

- **Python**: 3.11+
- **Node.js**: 18+
- **MongoDB**: 6.0+
- **操作系统**: Windows / Linux / Mac

## 🛠️ 快速安装

### 1. 克隆项目
```bash
git clone <repository-url>
cd easy-book
```

### 2. 启动MongoDB
```bash
# 使用Docker启动MongoDB
docker run -d --name mongodb -p 27017:27017 mongo:6

# 或启动本地MongoDB服务
mongod
```

### 3. 启动后端服务
```bash
cd backend

# 安装Python依赖
pip install -r requirements.txt

# 启动后端服务 (端口8002)
python -m api_server.main
```

后端服务将在 http://localhost:8002 启动
API文档: http://localhost:8002/docs

### 4. 启动前端服务
```bash
# 新开一个终端窗口
cd frontend

# 安装Node.js依赖
npm install

# 启动前端开发服务器 (端口5173)
npm run dev
```

前端应用将在 http://localhost:5173 启动

## 🎯 验证安装

访问 http://localhost:5173 确认系统正常运行：
- 看到学员管理页面
- 能够添加新学员
- 系统功能正常响应

## ❗ 常见问题

### MongoDB连接问题
确保MongoDB服务正常运行：
```bash
# 检查MongoDB状态
mongo --eval "db.adminCommand('ismaster')"
```

### 端口冲突
如果端口被占用，请修改配置或结束占用进程：
```bash
# Windows
netstat -ano | findstr :8002
taskkill /F /PID <进程ID>

# Linux/Mac
lsof -i :8002
kill -9 <进程ID>
```

### 依赖安装问题
使用国内镜像源：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/
```
