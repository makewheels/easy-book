# Easy Book - 泳课学员管理系统后端设计文档

## 1. 项目概述

Easy Book后端采用FastAPI + MongoDB架构，提供轻量级、高性能的RESTful API服务，支持学员管理、预约管理、签到扣费等核心功能。

## 2. 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| FastAPI | 0.104+ | Python现代Web框架 |
| MongoDB | 6.x+ | NoSQL文档数据库 |
| PyMongo | 4.x+ | MongoDB官方Python驱动 |
| Pydantic | 2.x+ | 数据验证和序列化 |
| Uvicorn | 0.24+ | ASGI服务器 |

## 4. 数据模型设计

### 4.1 学员模型 (Student)

```python
from datetime import datetime
from typing import Optional
from pydantic import BaseModel, Field
from bson import ObjectId

class StudentModel(BaseModel):
    """学员数据模型"""
    name: str = Field(..., min_length=1, max_length=50, description="姓名")
    nickname: Optional[str] = Field(None, max_length=50, description="别称")
    learning_item: str = Field(..., description="学习项目")
    package_type: str = Field(..., regex="^(1v1|1v多)$", description="套餐类型")
    total_lessons: int = Field(..., gt=0, description="总课程数")
    remaining_lessons: int = Field(..., ge=0, description="剩余课程数")
    price: int = Field(..., gt=0, description="售价(元)")
    venue_share: int = Field(..., ge=0, description="游泳馆分成(元)")
    profit: int = Field(..., ge=0, description="利润(元)")
    note: Optional[str] = Field(None, max_length=500, description="备注")
    create_time: datetime = Field(default_factory=datetime.utcnow)
    update_time: datetime = Field(default_factory=datetime.utcnow)

    class Config:
        json_encoders = {
            ObjectId: str
        }
```

## 6. 业务逻辑设计

### 6.1 预约冲突检查

```python
async def check_appointment_conflict(student_id: str, date: date, time_slot: str) -> bool:
    """检查预约冲突"""
    # 获取学员信息
    student = await student_service.get_by_id(student_id)
    
    # 如果是1v1课程，检查该时间段是否已有其他1v1预约
    if student.package_type == "1v1":
        existing = await appointment_repository.find_by_date_time(date, time_slot)
        for apt in existing:
            other_student = await student_service.get_by_id(apt.student_id)
            if other_student.package_type == "1v1" and apt.status != "cancelled":
                return True
    
    return False
```

### 6.2 签到扣费逻辑

```python
async def process_checkin(student_id: str, appointment_id: str):
    """处理签到扣费"""
    student = await student_service.get_by_id(student_id)
    
    if student.remaining_lessons <= 0:
        raise ValueError("剩余课程不足")
    
    # 扣减课程
    lessons_before = student.remaining_lessons
    lessons_after = lessons_before - 1
    
    # 更新学员剩余课程
    await student_service.update_lessons(student_id, lessons_after)
    
    # 创建上课记录
    attendance = AttendanceModel(
        student_id=student_id,
        appointment_id=appointment_id,
        attendance_date=datetime.now().date(),
        time_slot=appointment.time_slot,
        status="checked",
        lessons_before=lessons_before,
        lessons_after=lessons_after
    )
    
    return await attendance_repository.create(attendance)
```

### 6.3 数据统计服务

```python
class StatisticsService:
    """统计服务"""
    
    async def get_student_statistics(self, student_id: str):
        """获取学员统计信息"""
        student = await student_service.get_by_id(student_id)
        attendances = await attendance_repository.find_by_student(student_id)
        
        total_checked = len([a for a in attendances if a.status == "checked"])
        total_absent = len([a for a in attendances if a.status == "absent"])
        
        return {
            "student_info": student,
            "total_attended": total_checked,
            "total_absent": total_absent,
            "attendance_rate": total_checked / len(attendances) if attendances else 0
        }
```

## 8. 错误处理

### 8.1 自定义异常

```python
from fastapi import HTTPException

class BusinessException(Exception):
    """业务异常"""
    def __init__(self, message: str, code: int = 400):
        self.message = message
        self.code = code
        super().__init__(self.message)

class StudentNotFoundError(BusinessException):
    """学员不存在异常"""
    def __init__(self, student_id: str):
        super().__init__(f"学员不存在: {student_id}", 404)

class AppointmentConflictError(BusinessException):
    """预约冲突异常"""
    def __init__(self, time_slot: str):
        super().__init__(f"时间段 {time_slot} 已有1v1预约", 409)
```

### 8.2 全局异常处理

```python
from fastapi import Request
from fastapi.responses import JSONResponse

@app.exception_handler(BusinessException)
async def business_exception_handler(request: Request, exc: BusinessException):
    return JSONResponse(
        status_code=exc.code,
        content={
            "code": exc.code,
            "message": exc.message
        }
    )
```

from motor.motor_asyncio import AsyncIOMotorClient
from .config import settings

class Database:
    client: AsyncIOMotorClient = None
    database = None

db = Database()

async def connect_to_mongo():
    db.client = AsyncIOMotorClient(settings.MONGODB_URL)
    db.database = db.client[settings.DATABASE_NAME]
