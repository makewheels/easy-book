# Easy Book - 泳课学员管理系统

> 🏊‍♂️ 一个轻量级的泳课学员管理系统，专为个人使用设计。

## 📖 项目组成

本项目由以下几个主要部分组成：

- **[📋 项目文档](./doc/README.md)** - 完整的项目文档和开发指南
- **[🧪 测试项目](./tests/README.md)** - 独立的自动化测试套件
- **[backend/](./backend/)** - FastAPI后端服务
- **[frontend/](./frontend/)** - Vue.js前端应用

## 🚀 快速开始

### 环境要求
- Python 3.11+
- Node.js 18+
- MongoDB 6.0+

### 安装与运行
- **🚀 快速开始**: 查看 [安装指南](./doc/04-部署运维/安装指南.md) 搭建本地开发环境
- **🖥️ 生产部署**: 查看 [部署文档集合](./doc/04-部署运维/) 部署到服务器
- **⚙️ 数据库配置**: 查看 [MongoDB配置指南](./doc/04-部署运维/MongoDB配置指南.md) 配置数据库

### 核心功能
- 📱 移动端友好设计
- 👥 学员管理（增删改查）
- 📅 预约管理（创建、查询、冲突检查）
- ✅ 考勤管理（签到、课程统计）
- 🗓️ 表格日历（周视图、状态显示）
- 🚫 周一闭馆限制
